Bait

This is a project from Graphic Design III class in Old Dominion University. The task was to create a new graphic symbol using the provided letterform, 'B', and the word, "bait".

I explored various solutions for "bait" which range from the definitions "to entice" to the old blood sport, "bear baiting". Ultimately, the concept of the Anglerfish proved the strongest among the thumbnail sketches. Variations of it were refined more during the Roughs phase.
